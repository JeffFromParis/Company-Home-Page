Notre mission

Tout part d'un rêve. Un rêve dans lequel les femmes ne seraient plus considérées comme des objets. Un rêve dans lequel les hommes n'auraient pas dépenser la moitié de leur salaire pour un rendez-vous sans même savoir s'ils vont niquer. Mieux, un rêve dans lequel ils se font payer pour être sucés. Un rêve dans lequel les femmes qui ont l'impression d'être aux commandes de leur sexualité. Un rêve dans lequel les hommes se font sucés et en ils se font payer pour ça. Ce rêve est devenu réalité. Ce rêve, c'est Chibreur Privé.
